GP é um Módulo Gerenciador de Produtos

Este gerenciador faz parte dos planos de implementação no P.I. de nosso grupo, mas que possivelmente não será incluso por decorrência do prazo de entrega do projeto completo.

O DiBRe Inn é o sistema que estamos desenvolvendo no Projeto Integrador do 3º semestre pelo curso Tecnologia em Análise e Desenvolvimento de Sistemas do Centro Universitário SENAC Santo Amaro - SP, ano 2015.

Este módulo não dispõe do mesmo estilo aplicado no Projeto Integrador, pois lá utilizamos o framework Bootstrap 3 e aqui o visual e estilo foram desenvolvidos por nós mesmos, apenas contando com frameworks simples e leves:

  • Fonte Roboto, por Google no Google Fonts
  • Menu Responsivo com CSS3 puro, por Palloi Hofmann no Tableless
  • Cropbox, por Nguyen Hong Khanh no GitHub

Então o Gerenciador de Produtos faria parte do projeto DiBRe Inn, mas infelizmente não poderá ser aplicado.
